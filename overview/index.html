Web Development


Getting Started


To begin getting into web development, we must first understand the different systems that make it possible. These include three main programming languages that work in tandem to create a functioning webpage.

HTML


HTML is the most important language for a webpage, as organizes all the text elements and containers, and tells the webpage where to find all of its files. Without an HTML file, there is no webpage.

CSS


CSS, put simply, is what makes the site look pretty. CSS tells the website where to put everything and how it looks. It can also define the containers that HTML uses to organize things.

JavaScript


JavaScript can be looked at as the logic behind a website. It does the more complex functions that CSS and HTML can't do. Think about things like button functions, moving navigation bars, and data manipulation.
